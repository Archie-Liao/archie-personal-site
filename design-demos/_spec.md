# 设计 spec — 廖智强Archie 个人网站首页（3 版方向探索的共同输入）

> 三个版本**内容、约束、素材完全一致，只有视觉方向不同**，方便横向对比。每个 subagent 必须严格遵守本文件。

## 这是什么
为内容创作者「廖智强 Archie」做的个人网站**首页**高保真设计稿（单个 HTML，1440px 宽，用于桌面 1440×900 截图）。网站定位＝"视频日记 + 网络优质内容（当前以 AI 为主）"的沉淀归档。

## 受众 / 场景 / 目的
- 受众：B站/小红书粉丝、同频学习者、潜在合作方、作者本人
- 场景：搜某主题知识 / 了解·联系作者 / 回顾某期日记
- 目的：强第一印象 + 最新·精选内容入口 + 立住品牌气质

## 气质关键词（务必命中）
精致、复古、清新明朗、文气、有时间感、克制、安静但信息分明。
参照 Anthropic「Claude Fable 5」发布视觉：**奶油纸底 + 复古博物（蝴蝶/蛾/植物）版画 + 优雅衬线**。
**绝不**：暗黑底、高饱和、赛博霓虹、紫色渐变、emoji 图标。

## 硬约束（所有版本都要守）
1. **只做浅色**：暖奶油纸底（#F6F0E2 / #F5F0E8 一类），不要纯白冷灰，不要暗色
2. **字体经 fonts.loli.net 国内镜像加载**（Google 直连在中国大陆会失败→回退系统宋体）。务必用这一行：
   ```
   @import url('https://fonts.loli.net/css2?family=Playfair+Display:ital,wght@0,500;0,600;0,700;0,900;1,500;1,600&family=Noto+Serif+SC:wght@500;600;700;900&family=Inter:wght@400;500;600;700&family=Noto+Sans+SC:wght@300;400;500;700&display=swap');
   ```
   - 英文/数字大标题 = Playfair Display；中文标题 = Noto Serif SC（思源宋体）；正文 = Inter + Noto Sans SC（思源黑体）
3. **衬线标题 + 无衬线正文**搭配；**小字也必须清晰可读**（足够对比 + 字距）
4. **低饱和**：墨黑文字 + 单一暖 accent（墨绿 #3C5A4A 或 赭石 #B5612F），克制
5. **纸张质感**：极淡颗粒/做旧（SVG 噪点 data URI 叠 multiply，opacity ≤ 0.06）
6. **复古博物图用下方真图**，不要 CSS 画蝴蝶；克制、服从内容
7. 反 AI slop：无紫渐变 / 无 emoji 图标 / 无圆角卡片+左彩色边 / 无 SVG 画人画物 / display 不用 Inter 或系统字
8. **单个自包含 HTML 文件**；除 loli.net 字体 + 本地图片外**无任何外部资源**（不要 Unsplash、不要 Google 直连）
9. 用下方**真实内容**，不要 Lorem；1440px 设计宽，针对 1440×900 首屏优化
