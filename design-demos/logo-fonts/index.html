怎么选 / 怎么混：
  • A — 要明显爨体/碑刻文气，接受 Logo 更「字标化」、个性更强。
  • B — 要古印刷、老铅字，但保持正常字比例，最贴出版物站点气质。
  • C — 要字体2 那种修长压扁木刻标题感；nav 小字需单独调比例。
  • 混搭 — 例如「B 的字 + C 的压扁程度一半」或「A 的古意 + B 的暖墨色值」。
  • 仍选原版 — 也可以；那就只调暖墨/字距，不换字体 family。
