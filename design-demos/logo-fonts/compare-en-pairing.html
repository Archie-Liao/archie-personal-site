Logo 字体 v3 · 中英搭配

你反馈刀隶体与英文 Playfair 不搭——这判断准确：刀隶是方笔碑刻（隶楷），Playfair 是欧洲 Transitional 衬线斜体，属于不同「字体文明」。 本轮 3 个候选都改走宋/衬线系，与 Archie 英文同屏时更协调；英文固定不动。

打开方式（重要）：请用本地服务器，不要直接双击 file://。 cd design-demos && python -m http.server 8765 → http://127.0.0.1:8765/logo-fonts/compare-en-pairing.html

现在就能看，不必先补信息。 若要缩小范围，可选补充：
  • 中文气质：要更「古印刷铅字」还是更「现代出版物克制」？（默认两者都给你看了）
  • 英文是否可换：若 D/E/F 中文满意但 Archie 仍别扭，可第二轮试 Cormorant / Libre Baskerville（默认本轮不换）
  • 参考图：任意中英 lockup 截图 / 网站链接（比文字描述快）
  • 否决项：例如「不要仿宋」「不要过细」「不要像政府网站」
