# 个人网站设计思路与需求模版

> 用途：在改版或重做个人网站之前，先想清楚「要什么、不要什么」。  
> 填完后把文档发给我，我可以据此整理站点地图、页面布局和改版计划。  
> 你不需要懂 CSS 或组件名，用「访客视角」描述即可。

---

## 一、先搞懂：个人网站在解决什么问题

不用先想「好不好看」，先回答这三句话（能写多短写多短）：

| 问题 | 你在想什么 |
|------|------------|
| **我是谁 / 做什么** | 例：日更视频日记的内容创作者 |
| **访客是谁** | 例：B 站粉丝、同频学习者、潜在合作方 |
| **他们来这里要干什么** | 例：看某一期沉淀、搜知识点、了解我、联系我 |

### 一句话定位（填空）

> 这是一个给 **[谁]** 看的网站，用来 **[主要目的]**，让他们能 **[完成的一件事]**。

**当前项目大致接近：**

> 给 **粉丝和学习者** 看的网站，用来 **沉淀每期视频日记**，让他们能 **快速回顾 AI 总结、知识卡片和字幕**。

若这句话和你真实想法不一致，应先改定位，再改页面。

## 二、个人网站通常有哪些基本页面

不是越多越好。内容型个人站常见组合如下：

| 页面 | 干什么 | 你是否需要 |
|------|--------|------------|
| **首页** | 第一印象 + 最新/精选内容入口 | 几乎都要 |
| **内容列表** | 所有文章/视频日记归档 | 内容多时需要 |
| **单篇详情** | 一期内容的完整阅读体验 | 核心页 |
| **关于我** | 你是谁、为什么做、怎么联系 | 几乎都要 |
| **专题/系列** | 按主题串起来（如「决策系统」系列） | 可选 |
| **搜索** | 搜标题、字幕、知识点 | 内容多了很有用 |
| **订阅/关注** | 引导去 B 站、Newsletter 等 | 可选 |
| **作品集** | 若你还有设计/开发项目 | 视身份而定 |

### 视频日记类网站的合理结构（3～5 页即可）

```
首页 → 挑重点展示
  ↓
日记列表（可和首页合并，也可分开）
  ↓
单期详情（视频 / AI 总结 / 卡片 / 字幕）
  ↓
关于我（人设 + 外链）
```

## 三、每一页怎么描述布局（不用懂 CSS）

每一页按下面 **5 个问题** 描述即可，我会翻译成具体布局和组件。

1. **第一眼看到什么**（最上面、最大的一块）
2. **往下还有什么**（第二屏、第三屏各一块）
3. **每块里放什么内容**（文字 / 图 / 视频 / 列表）
4. **希望访客做什么**（点进详情 / 去 B 站 / 联系我）
5. **感觉像什么**（参考站或关键词：干净、杂志感、深色沉浸…）

## 五、对照当前网站时的检查清单

打开本地预览（`npm run dev` 后访问 http://localhost:5173/），按下面维度标出不满意的地方：

| 维度 | 可以问自己 |
|------|------------|
| **信息架构** | 首页信息太多还是太少？详情用 Tab 还是长页滚动更顺眼？ |
| **文案** | Hero、按钮、关于页是不是太像 AI 套话？ |
| **视觉** | 深色是否太压抑？主色是否不喜欢？ |
| **卡片** | 封面+摘要是否像「通用博客」而不是「你的频道」？ |
| **品牌** | 名字露出够不够？整体像不像你？ |
| **转化** | 「去 B 站」够不够显眼？联系入口有没有？ |
| **内容密度** | 知识卡片、字幕是否一打开就 overwhelm？ |

## 六、收集信息的最小清单

不必一次凑齐，有下面这些就可以开工一版「像你的」网站：

1. **一句话定位**（第 1 节）
2. **3 个参考网站链接** + 各写一句喜欢什么
3. **首页从上到下** 想放几块、每块一句话
4. **详情页** 要 Tab 还是长页滚动
5. **关于我** 一段真自我介绍（口语也行）
6. **视觉**：深色/浅色 + 喜欢/讨厌的颜色
7. **现在这版最讨厌的 3 点**

封面、每期真实内容可以后补；结构和对齐风格可以先定。

## 七、详情页两种常见方案（对照思考）

### 方案 A：Tab 切换（当前项目）

- **适合**：每期内容块多，但不想页面过长
- **缺点**：字幕和总结不能同时扫一眼

### 方案 B：长页滚动

- **适合**：想营造「一期是一篇完整文章」的阅读感；分享、打印更直观
- **结构示例**：视频 → 本期概述 → 要点 → 金句 → 知识卡片 → 字幕

在模版 **第 5 节** 勾选你更倾向的方案即可。
