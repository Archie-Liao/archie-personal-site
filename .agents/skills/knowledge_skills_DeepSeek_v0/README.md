# knowledge_skills v2.0 使用说明

## 文件结构

```
knowledge_skills_ChatGPT_v0/
├── SKILL.md       ← 核心规则文件（最高优先级）
└── README.md      ← 本文件
```

## 调用方式

在对话中直接使用：`/knowledge_skills_ChatGPT_v0` 或提及「处理笔记」「搭建知识库」等关键词。

## 核心流程（3 阶段 · 2 次确认）

1. **接收与存档**（Inbox → Raw）：原文存入 raw/ 层 → **确认点①**
2. **拆分与分类**（Raw → Wiki）：拆分条目 + 打标签 + 生成 wiki 草稿
3. **归档与聚类**（Wiki → Index）：创建页面 + 双向链接 + 更新索引 → **确认点②**

## v2.0 vs v1.0 变化

| 维度 | v1.0 | v2.0 |
|------|------|------|
| 规则约束 | 无红线 | 4 条核心红线 |
| 流程步骤 | 6 步强制线性 | 3 阶段灵活执行 |
| 人工确认 | 每步确认（6 次） | 关键节点（2 次） |
| 标签体系 | 5 个随意标签 | 17 个三分类标签 |
| Frontmatter | 散落无规范 | 统一必填 + 选填 |
| 依赖 | Python 脚本 + JSON 输出 | 纯 LLM 直出 Markdown |
