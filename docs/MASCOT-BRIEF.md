### 基础款 · 即梦提示词（整段复制 · 可垫 B3）

```text
一只天蓝潮玩吉祥物小宠，全身站立，纯白棚拍背景。身体为3D磨砂塑料质感，柔和棚灯光，身体表面无毛发。
头身比例：超大圆头、短胖小身子、几乎没有脖子，Q版潮玩。
颜色：身体与头通体干净天蓝；两只大竖耳内侧浅桃粉。
五官：
1）一对大白眼、小黑瞳，眼型向内下压，不耐烦傲娇凶萌；
2）眉毛是本角色最突出的识别特征，必须与身体质感形成强烈对比，绝不能和身体同为磨砂蓝塑料：一对橙红色真实毛发眉毛，能看清一缕一缕的发丝走向；眉尾呈火焰形状——向外上扬、末端像火舌分叉跳跃，形状明确醒目；禁止整块毛毡、禁止釉片硬边、禁止金属、禁止同色塑料浮雕眉；不要粉嫩细弯眉；
3）极小鼻尖；短下撇嘴线。
耳朵：大尖竖耳，外缘缺口咬痕，双耳完整。
四肢（形态锁定）：
- 每只手正好四根手指：接近正常人的短手指，有清晰指节与指腹，指尖略收，禁止圆滚滚香肠指，禁止三指或五指；
- 每只脚正好四根脚趾：恐龙脚趾感，略粗壮、趾端略尖或带小爪感，有厚度层次，禁止圆球趾，禁止三趾或五趾。
姿势：自然站立，短臂略张；无披风、无键帽、无兵器、无围巾、无衣服。
气质：傲娇精明；不要甜美粉嫩女性化；不要史迪仔；不要蓝龙布鲁（鹿角红围巾）。
```

验收：①橙红缕状毛眉 + 火焰眉尾；②4 指人手形 / 4 趾恐龙感；③裸身无斗篷键帽。

## 变体 · 斗篷 + 键帽（皮肤 / 动画 · **未删 · 默认定稿后做**）

> **不是默认长相。** 默认 = 裸身基础款。本段是加戏：可选皮肤，或回复/发功时的动画态。  
> **已废**：披风 + 笔状长枪。

| 项 | 设定 |
|----|------|
| 披风 | **红或橙系**短披风（朱红 / 橙红 / 暖橙），布感简洁武侠侠客感；不要粉嫩；不要花纹龙袍 |
| 兵器 | **无**长枪 / 笔枪 / 任何兵器 |
| 键帽 | 身周环绕一串**立体键盘键帽**（漂浮键环），清晰可辨，有远近层次 |
| 姿势 | 双手**弹压打字**，指尖按空气中的键帽，像武侠弹琴催动音波发功；有力量感与节奏感，不要呆呆摊手 |
| 气质词（仅变体） | 侠客感（键帽音波 / 弹压发功） |
| 身体锁点 | 与已定稿基础款一致（含异质感眉、4 指人手形、4 趾恐龙感） |

### 皮肤/动画 · 即梦提示词（垫已定稿基础款 · 整段复制）

```text
严格以参考图角色为准：同一天蓝潮玩小宠，巨头巨眼，已定稿的异质感眉毛，大竖耳外缘缺口，小身子，每手四指（近人手形、有指节），每脚四趾（恐龙趾感），3D磨砂塑料身体，白底棚拍。
只改姿势与配件，五官头身与指趾数不变：
1）披一件短披风，颜色用红色或橙色系列（朱红、橙红、暖橙任一干净饱和色），布感简洁有武侠侠客感，不要粉嫩，不要花纹龙袍；
2）取消任何长枪/笔枪/兵器；
3）身周环绕一连串立体键盘键帽（像漂浮的键环），键帽清晰可辨，有远近层次；
4）双手抬起作弹压打字的动作，指尖按下空气中的键帽，姿势像武侠弹琴发功、催动音波——有力量感与节奏感，不要呆呆摊手；仍保持每手四指。
站立或微微含势全身，双耳完整。保持不耐烦傲娇眼神与参考图同款眉毛。不要史迪仔，不要蓝龙布鲁鹿角红围巾。
```

出图后只评：披风色、键帽环、弹压「发功」感——且标为 **skin/动画素材**，不替换默认全身。
