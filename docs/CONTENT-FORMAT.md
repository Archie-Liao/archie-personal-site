# 站内内容文案与排版规范

> **写 `posts.ts`、洗字幕、改 `/posts` 预览前读本文。**  
> 批次进度另见 [`P2-CONTENT-INBOX.md`](P2-CONTENT-INBOX.md)（管「做到哪了」，不管断句细则）。  
> **详情思维导图**（中心/节点抽取、布局、自检）→ [`MINDMAP-RULES.md`](MINDMAP-RULES.md)，不在本文重复。

| 章 | 管什么 |

## §A · `date` 怎么定

| 类型 | 优先级（取第一个有的） |
|------|------------------------|
| **视频日记** | 字幕开场「今天是…年…月…号/日」→ `YYYY-MM-DD` |
| **外链** | ① 页面上写明的**发布时间**（须核对截图/原文，勿猜） ② Archie **要求上传当天** |
| **禁止** | 随手假日期（如曾用的 `2025-06-*` mock） |

**例**：今日头条 Dan Koe 文标题下 **`2026-04-01 19:36`** → `date: "2026-04-01"`。

显示（只改数据，格式不变）：

| UI | 格式 |
|----|------|
| 月分组 | `YYYY.MM` |
| 行内 | `MM-DD · 标签` |
| VOL | `VOL.YYYY.MM.DD · 平台/期数` |

## §B · 标题与 AI 总结

| 规则 | |
|------|--|
| **有原文标题** | **必须原样用**（核对页面；禁止凭 URL 臆造） |
| **无标题** | AI 概括，短、可检索 |
| **摘要分段** | `overview` 用 `\n\n`；段勿巨长、勿句句一段；**段首概括本段** |
| **加粗** | 总结句/关键词 `**…**`（`renderInlineMarks`） |
| **要点** | `keyPoints` 短句 |
| **金句** | `quote` 必须是可引用短句；禁止「待补本期金句」占位上站 |
| **详情长总结** | 首段=导读（短橙条只贴导读）；其后展开段 → 要点卡 → 金句 → 辅区（关键词/检查）降权 |

## §C · 列表预览（延展 · 栏宽）

### 阅读延展

| 规则 | |
|------|--|
| **用途** | 只补主摘要**没写过**的短注 |
| **禁止** | 再贴 `overview` / `keyPoints` |
| **篇幅** | ≤2 段 · 每段 ≤120 字 · 合计 ≤220 字 |
| **可无** | 无新信息 → 不显示 |

实现：`src/app/utils/postsPreviewExtended.ts`。

### `/posts` 栏宽与预览字号

| 规则 | |
|------|--|
| **左栏** | 索引（紧凑行高） |
| **中栏** | 预览 · 主视觉（最宽） |
| **右栏** | 导语矮卡（最窄 · 不拉满） |
| **跟栏** | 预览卡 `max-width` 跟中栏（现约 **52rem**）；标题/正文/摘录同步 |
| **点选对齐** | 桌面：**三栏锁视口**；中/右内滚；换篇只右栏 `scrollTop=0`；标题 hover 微右移+色条 |
| **实现** | `Layout` `layout--posts` · `posts-list.css` · `PostsListPage` |

## §D · 字幕稿排版

> AI 处理 `design-demos/视频日记字幕稿/` → `public/subtitles/` 时必须遵守本节。  
> 改规则时同步改 `scripts/format-subtitles.mjs`。

### 来源

- 源：`design-demos/视频日记字幕稿/{期数}：{标题}.txt`（**期数只用开头数字**，勿加 `Day`）
- 发布：`public/subtitles/ep{NNN}.txt`

### 可以改 / 不可以改

| 可以 | 不可以 |
|------|--------|
| 标点、断句拼接、按规则分段 | 改用字用词（除 ≥80% 确定的识别错字） |
| 保留重复口语只加标点 | 恢复时间轴 |

### 分段（落地）

输出：**一行 = 一个 `<p>`**（段间空行）。

1. **句号分段（默认）**：`。！？` 收束处断开  
2. **主题段另起**：日期开场、`感谢…`、`终身学习` / `与君共勉` / `明天见`  
3. **超长句**：单句 >120 字且含 `，；` 时，可在从句边界再拆  
4. **禁止**：整篇 1～2 大段；按固定字数机械切

### 句内标点（脚本）

相邻导出行默认 `，`；换意群 `。`；末句无句号补 `。`。

### 展示与工作流

- `SubtitleViewer`：空行 → 新 `<p>`；浏览器不二次断句  

```powershell
node scripts/format-subtitles.mjs
npm run dev   # → /post/ep099
```
