### Design System（v3 定稿）

**Token（从 benchmark 抽，非 recipe hex）**

| Token | 值 | 说明 |
|-------|-----|------|
| `--paper` | `#F5F0E8` | ground（与 benchmark 一致） |
| `--paper-2` | `#EFE7D8` | 窄条 / band |
| `--ink` | `#191919` | 正文 |
| `--accent` | `#CC785C` | 主 accent（production 同源） |

| `--signal` | `#C4A035` | 第三对比（可选），仅标签/eyebrow，≤5% 屏 |
| `--rule` | `rgba(25,25,25,0.14)` | hairline |

**Typography**

- Folio / issue：`Playfair Display` 900，Hero 与 stat-band
- 中文标题：`Noto Serif SC` 600–700
- 正文：`Inter` + `Noto Sans SC`
- Turley 张力：Hero **一组**全大写 eyebrow（小字 Barlow/Inter condensed），**不**全文 screaming caps

**Layout（v0 范围）**

- 顶栏：纸底 + hairline 底边（**无**黑顶栏）
- Hero：左文右「版画框」或 folio 数字叠压；**一条** punch band（高 ≤120px）
- 时间线：表格式 index（folio 数字列 + 标题列 + 日期列）
- 精选：3 卡，**禁止**整卡大面积变色（D1 pick 教训）

**Motion 菜单（v0 须可试吃，≥3）**

1. Hero folio 数字 **slam-in**（`cubic-bezier(0.22, 1, 0.36, 1)` 快入，200ms）
2. Index row hover **snap** 左移 + punch 左条（非 smooth fade）
3. Scroll reveal：section **step** 入场（stagger 60ms）
4. （可选 Tweaks）微 misregistration 1–2px on punch block

**C 层破例（≤3，每条面积上限）**

1. Hero 底 punch band — ≤12% 视口高
2. Issue badge 小色块 — ≤4% 面积
3. Index row hover 左条 — ≤3px 宽，非全行铺色

**BorrowableParts（≥3 · v0 头注释）**

- 表格式时间线索引 + folio 数字列
- Hero slam folio + punch band（局部 Turley 张力）
- hairline + 纸底上的编辑型 eyebrow 标签
