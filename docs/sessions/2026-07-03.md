## 7. project-context Skill v2（通用项目治理）

> 2026-07-03 晚间收工 · 权威源 `.agents/skills/project-context/`

### 背景

Archie 要求从本仓 AGENTS / rules / 宪章 / COMPLEX-TASK / MULTI-SOURCE 抽象**可复用到其他项目**的 Skill；执行时对齐 Cursor `create-skill` 规范（frontmatter、≤500 行、references 一层深）。

### 交付

| 项 | 路径 |
|----|------|
| 主流程 | `.agents/skills/project-context/SKILL.md`（流程 0–5） |
| 模板 | `references/`：宪章、门控、多源、doc 生命周期、规则决策树 |
| Bootstrap | `scaffold.mjs --with-governance` |
| 索引 | `AGENTS.md` 文档索引一行 |

### 关键原则（写入 Skill）

1. 仅 `AGENTS` + `alwaysApply` mdc 自动注入；宪章/专题须 gotchas 一行索引  
2. 通用规则 → 宪章；路径/案例 → 专题 doc（如 MULTI-SOURCE）  
3. 多源 ≥2 权威文件 → `## 执行计划` 优先级表  
4. 复杂任务 → 首条 `## 复述需求`  
5. `{EXPLORE_DIR}` vs `{APP_DIR}` 分离探索与生产  

### 复用

复制整个 `.agents/skills/project-context/` 到新仓库 → `scaffold.mjs --with-governance` → 填 gotchas。
